# Peta Cerita (PetCer)

PetaCerita is a collaborative travel planning application for organizing itineraries, collecting destination ideas, viewing daily routes, managing budgets, and splitting expenses with friends in one shared trip space.

Website: [peta-cerita.vercel.app](https://peta-cerita.vercel.app)

## Key Features

- Create and manage collaborative trips.
- Invite members through a shareable invitation link.
- Save destination ideas from Google Maps.
- Organize daily itineraries with drag and drop.
- View maps, routes, distances, and travel warnings.
- Open the original Google Maps link for each destination.
- Plan budgets by category.
- Record and split bills by item.
- Calculate balances between trip members automatically.
- Confirm completed payments from both parties.
- Generate downloadable and shareable travel cards.
- Keep trip information updated in real time.

- [Zod](https://zod.dev/) for data validation
- [Vitest](https://vitest.dev/) and Playwright for testing
- [Vercel](https://vercel.com/) for deployment

## Design

PetaCerita uses a Doodle Notebook design system with handwriting-inspired typography, notebook grids, highlighter accents, sketch-like lines, and imperfect card and button shapes. The interface is responsive across desktop and mobile devices.

## Product Focus

PetaCerita helps groups plan trips in a more organized way without separating itineraries, maps, notes, budgets, and shared expenses across multiple applications.

## Project Status

PetaCerita is under active development, with ongoing improvements to its features and user experience.
